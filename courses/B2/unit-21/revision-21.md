# Revision – Unit 21

## 🎯 Lesson Objective

By the end of this lesson, the student will be able to:
- describe and analyse security systems precisely  
- evaluate advantages and risks  
- understand authentic listening about airport security  
- express balanced and analytical opinions  

---

## 1. Vocabulary Precision 

Choose the BEST option:

1. Security measures are designed to __________ potential threats.  
   (prevent / avoid / stop)

2. The system aims to __________ suspicious behaviour early.  
   (detect / identify / recognise)

3. Airports are trying to __________ new types of risk.  
   (respond to / deal with / manage)

4. The approach is becoming more __________ rather than standardised.  
   (flexible / tailored / adjusted)

5. Increased security may create new __________ in the system.  
   (weaknesses / gaps / vulnerabilities)

## 3. Reduced Clauses (Accuracy)

Choose the BEST option:

1. Passengers __________ through checkpoints may face delays.  
   (moving / moved / having moved)

2. Systems __________ to prevent attacks are constantly updated.  
   (designed / designing / having designed)

3. People __________ suspicious behaviour may be stopped.  
   (showing / shown / being shown)

4. Security measures __________ yesterday are still in place.  
   (introduced / introducing / having introduced)

5. Data __________ from different sources improves detection.  
   (collected / collecting / having collected)

---

## 4. Precision in Context 

Choose the BEST option:

1. Security is based on multiple __________ rather than a single system.  
   (levels / layers / stages)

2. Authorities are not __________ to reveal all measures.  
   (allowed / able / willing)

3. The goal is to make attacks more __________ to carry out.  
   (difficult / complicated / challenging)

4. Some countries introduced __________ security after past incidents.  
   (extreme / strict / intensive)

5. Systems must __________ to new threats over time.  
   (adapt / adjust / change)
