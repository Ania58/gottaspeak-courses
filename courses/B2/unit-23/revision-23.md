# Revision – Unit 23

## 🎯 Lesson Objective

By the end of this lesson, the student will be able to:
- analyse awkward social situations  
- understand real-life advice  
- evaluate behaviour and strategies  
- express reactions and opinions naturally  

---

## 1. Vocabulary Precision 

Choose the BEST option:

1. Parents need to __________ expectations before the event.  
   (define / set / establish)

2. The situation can quickly __________ out of control.  
   (spiral / get / go)

3. It’s important to __________ a clear plan in advance.  
   (have / develop / make)

4. The event may __________ problems if not organised properly.  
   (lead to / cause / create)

5. Parents often feel __________ after hosting a party.  
   (worn out / tired / exhausted)

## 3. Precision in Context (Balanced Options)

Choose the BEST option:

1. It’s important to __________ expectations early.  
   (control / manage / organise)

2. The party can easily __________ too expensive.  
   (get / become / turn)

3. You should __________ a backup plan.  
   (arrange / set / prepare)

4. Parents need to __________ different situations.  
   (deal / manage / handle)

5. The situation may __________ if expectations are unclear.  
   (increase / worsen / develop)
