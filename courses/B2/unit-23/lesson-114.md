# Lesson 114 – Complaining (Soft vs Direct)

## 🎯 Lesson Objective

By the end of this lesson, the student will be able to:
- express complaints in a natural way  
- adjust tone depending on situation  
- avoid sounding too aggressive or too weak

## 3. Language Focus – Tone

👉 soft:

✔ It’s not ideal when…  
✔ I was wondering if…  
✔ The issue seems to be…  
✔ It’s a bit inconvenient  

---

👉 direct:

✔ This is unacceptable  
✔ There’s a serious problem  
✔ This needs to be fixed  
✔ I’m not happy with this  

---

## 4. Precision Task 

Choose the BEST option:

1. It’s __________ inconvenient when this happens.  
   (a bit / slightly / quite)

2. I was __________ if this could be resolved.  
   (thinking / wondering / asking)

3. This is __________ a problem that needs attention.  
   (clearly / definitely / seriously)

4. The issue __________ to be related to maintenance.  
   (appears / looks / seems)

5. I’m not __________ with the current situation.  
   (happy / satisfied / pleased)

## 9. Reading

Read the text:

👉 *Complaining is a natural part of communication, but the way it is expressed can have a significant impact on the outcome.*

*In some situations, a softer approach may be more effective. Phrases that reduce directness can help avoid conflict and encourage cooperation.*

*However, there are cases where being direct is necessary, especially when the issue is serious or urgent.*

*Understanding how to adjust tone allows people to communicate more effectively and achieve better results.*
