## 3. Grammar – Inversion

Normal:
- Authorities rarely stop these networks.

Inversion:
- Rarely do authorities stop these networks.

---

Normal:
- This trade is not only illegal, it is dangerous.

Inversion:
- Not only is this trade illegal, but it is also dangerous.

---

More examples:

- Never have we seen such a rapid increase in demand  
- Under no circumstances should people buy illegal products  
- Only then did authorities realise the scale of the problem  

---

## 4. Build Inversion

Rewrite:

1. Authorities rarely control these networks.  
2. Governments never react fast enough.  
3. This trade is not only illegal, it is harmful.  
4. People should never ignore the consequences.  
5. We have rarely seen such high demand.  

---

## 5. Sentence Transformation

Rewrite using inversion:

1. They realised the problem only later.  
2. We can understand the scale only then.  
3. This is not only a crime, it is a global issue.  
4. People should not buy these products under any circumstances.  
5. We have never seen such a situation before.  

---

## 6. Precision Through Emphasis

Rewrite to make the sentence stronger:

1. This problem is serious.  
2. Authorities react slowly.  
3. The situation is getting worse.  
4. People ignore the risks.  
5. The system is weak.  

Use inversion where possible.

## 9. Reading

Read the text.

Illegal trade continues to grow despite increasing regulation. Rarely do authorities manage to fully control networks operating across multiple countries. These organisations adapt quickly, changing routes and methods to avoid detection.

Not only is this trade illegal, but it is also highly organised. Groups involved in wildlife trafficking often operate in the same way as other criminal networks, making large profits while taking minimal risks.

Under no circumstances should such activities be ignored. However, weak enforcement and corruption allow many operations to continue without serious consequences. In some regions, officials turn a blind eye, allowing illegal goods to move freely.

Only when governments take coordinated action do these networks begin to weaken. Until then, the problem is likely to continue growing.
